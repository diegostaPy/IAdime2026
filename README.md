urso de Inteligencia Artificial - Facultad de Ingeniería (FIUNA)

Bienvenidos al repositorio oficial del curso de Inteligencia Artificial. Este espacio está dedicado a la gestión de materiales, clases prácticas, proyectos y recursos complementarios para el desarrollo del curso.
📚 Acerca del Curso
Objetivo General

Formar profesionales capaces de aplicar algoritmos basados en aprendizaje de máquina para resolver problemas desafiantes en los campos de las ciencias y la ingeniería.
Objetivos Específicos

    Desarrollar los conceptos teóricos fundamentales para la implementación de algoritmos de IA.

    Explicar y comparar las redes neuronales (tanto profundas como clásicas) con otros modelos de aprendizaje automático.

    Determinar cuándo una red neuronal profunda o un algoritmo inteligente es la opción más adecuada para un problema particular.

    Aplicar métodos y algoritmos de vanguardia para resolver problemas y desafíos concretos en ciencia e ingeniería.

    Demostrar la comprensión de los temas a través de un proyecto final práctico, cuyo código será alojado en este repositorio (GitHub).

💻 Proyecto Final

El proyecto final es una parte fundamental de la evaluación. Consiste en la aplicación de las técnicas aprendidas para resolver un problema real o simulado.

    Objetivo: Demostrar la comprensión integral del ciclo de vida de un proyecto de Machine Learning, desde el planteamiento del problema y la preparación de los datos hasta la implementación, evaluación y presentación de resultados.

    Formato: El proyecto debe ser subido a este repositorio de GitHub. Se espera un código limpio, bien documentado y un informe (puede ser un README.md en la carpeta del proyecto) que explique la solución.

    Temáticas: Clasificación, regresión, series temporales, o procesamiento de imágenes, entre otros.

    Mejoras: Durante los períodos de exámenes finales (1er Final y 2do Final), los estudiantes tendrán la oportunidad de presentar mejoras a sus modelos basándose en la retroalimentación recibida.
